### Diccionarios en Python
Los diccionarios son colecciones no ordenadas de pares clave-valor. Cada clave es √∫nica y se asocia a un valor.

#### Creaci√≥n de diccionarios
Puedes crear un diccionario utilizando llaves `{}` y separando los pares clave-valor con comas.

```python
# Creaci√≥n de un diccionario
persona = {
    'nombre': 'Alice',
    'edad': 25,
    'ciudad': 'Madrid'
}
```

#### Acceso a elementos
Puedes acceder a los valores de un diccionario utilizando su clave.

```python
print(persona['nombre'])  # Imprime el valor asociado a la clave 'nombre'
print(persona['edad'])    # Imprime el valor asociado a la clave 'edad'
```

#### Modificaci√≥n de elementos
Puedes modificar los valores de un diccionario asignando un nuevo valor a una clave espec√≠fica.

```python
persona['edad'] = 26
print(persona)  # Imprime el diccionario actualizado
```

#### Eliminaci√≥n de elementos
Puedes eliminar un par clave-valor de un diccionario utilizando la palabra clave `del`.

```python
del persona['ciudad']
print(persona)  # Imprime el diccionario sin la clave 'ciudad'
```

#### M√©todos de diccionarios
Los diccionarios en Python tienen varios m√©todos √∫tiles para agregar, eliminar y manipular elementos.

| M√©todo | Ejemplo | Descripci√≥n |
| ------ | ------- | ----------- |
| `keys()` | `persona.keys()` | Devuelve una lista con las claves del diccionario. |
| `values()` | `persona.values()` | Devuelve una lista con los valores del diccionario. |
| `items()` | `persona.items()` | Devuelve una lista de tuplas con los pares clave-valor. |
| `get()` | `persona.get('nombre')` | Devuelve el valor asociado a una clave. |

### Conjuntos en Python
Los conjuntos son colecciones no ordenadas de elementos √∫nicos.

#### Creaci√≥n de conjuntos
Puedes crear un conjunto utilizando llaves `{}` o la funci√≥n `set()`.

```python
# Creaci√≥n de un conjunto
colores = {'rojo', 'verde', 'azul'}
numeros = set([1, 2, 3, 4, 5])
```

#### Acceso a elementos
Debido a que los conjuntos no est√°n ordenados, no puedes acceder a elementos por √≠ndice. Sin embargo, puedes comprobar si un elemento est√° presente en un conjunto.

```python
print('rojo' in colores)  # True
print('amarillo' in colores)  # False
```

#### Eliminaci√≥n de elementos
Puedes eliminar un elemento de un conjunto utilizando el m√©todo `remove()`.

```python
colores.remove('verde')
print(colores)  # Imprime el conjunto sin el elemento 'verde'
```

#### Eliminaci√≥n de elementos duplicados
Los conjuntos son √∫tiles para eliminar elementos duplicados de una lista.

```python
numeros = [1, 2, 3, 4, 2, 3, 5]
numeros_unicos = set(numeros)
print(numeros_unicos)  # {1, 2, 3, 4, 5}
```

#### M√©todos de conjuntos
Los conjuntos en Python tienen varios m√©todos √∫tiles para agregar, eliminar y manipular elementos.

| M√©todo | Ejemplo | Descripci√≥n |
| ------ | ------- | ----------- |
| `add()` | `colores.add('amarillo')` | Agrega un elemento al conjunto. |
| `remove()` | `colores.remove('verde')` | Elimina un elemento del conjunto. |
| `clear()` | `colores.clear()` | Elimina todos los elementos del conjunto. |
| `union()` | `colores.union(numeros)` | Devuelve un nuevo conjunto con todos los elementos de ambos conjuntos. |
| `intersection()` | `colores.intersection(numeros)` | Devuelve un nuevo conjunto con los elementos comunes a ambos conjuntos. |
| `difference()` | `colores.difference(numeros)` | Devuelve un nuevo conjunto con los elementos que est√°n en el primer conjunto pero no en el segundo. |
