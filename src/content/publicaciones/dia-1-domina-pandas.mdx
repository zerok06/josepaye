## Estructuras de Datos Principales

### Series
Una estructura de datos unidimensional que representa una serie de valores con un etiquetado de √≠ndice.

```python
serie = pd.Series([1, 2, 3])
```
| Indices    | Data     |
|------|-------|
| 0    | 1       |
| 1    | 2       |
| 2    | 3       |

Ademas, puedes renombrar los √≠ndices de la serie:

```python
serie = pd.Series([1, 2, 3], index=['a', 'b', 'c'])
```



### DataFrames
Una estructura de datos bidimensional que representa una tabla con filas y columnas.

```python
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}
```
| A    | B     |
|------|-------|
| 1    | 4       |
| 2    | 5       |
| 3    | 6       |


## Creaci√≥n de DataFrames desde Archivos

Puedes crear un DataFrame a trav√©s de un archivo CSV o Excel. Para hacerlo, puedes usar la funci√≥n `read_csv()`, `read_excel()` o `read_json()` de Pandas.

```python
# Desde CSV

# Desde Excel

# Desde JSON

```
> Puedes explorar los otros m√©todos de lectura de archivos en la [documentaci√≥n oficial](https://pandas.pydata.org/docs/reference/api/pandas.read_csv.html).

## Atributos B√°sicos de los DataFrames

### Columnas
Puedes acceder a las columnas de un DataFrame utilizando el atributo `columns`:

```python
print(df.columns)
```

### Filas
Puedes acceder a las filas de un DataFrame utilizando el atributo `index`:

```python
print(df.index)
```

### Valores
Puedes acceder a los valores de un DataFrame utilizando el atributo `values`:

```python
print(df.values)
```

### Dimensiones
Puedes obtener el tama√±o de un DataFrame utilizando el atributo `shape`:

```python
print(df.shape)
```

### Tipos de Datos
Puedes obtener los tipos de datos de las columnas de un DataFrame utilizando el atributo `dtypes`:

```python
print(df.dtypes)
```

### Informaci√≥n del DataFrame
Puedes obtener informaci√≥n sobre el DataFrame utilizando el atributo `info`:

```python
print(df.info())
```
