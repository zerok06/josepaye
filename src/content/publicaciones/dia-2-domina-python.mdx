### Condicionales:

Los condicionales te permiten ejecutar bloques de c√≥digo solo si se cumple una condici√≥n espec√≠fica.

#### Condicion `if`
El condicional `if` se utiliza para ejecutar un bloque de c√≥digo si la condici√≥n es verdadera.

```python
# Ejemplo de condicional if
edad = 18

if edad >= 18:
    print("Eres mayor de edad.") 
```

#### Condicion `else`
El condicional `else` se utiliza para ejecutar un bloque de c√≥digo si la condici√≥n del `if` es falsa.

```python
edad = 18

if edad >= 18:
    print("Eres mayor de edad.")  # Se ejecuta si la condici√≥n es verdadera.
else:
    print("Eres menor de edad.")  # Se ejecuta si la condici√≥n es falsa.
```

#### Condicion `elif`
El condicional `elif` es una combinaci√≥n de `else` e `if`. Permite verificar m√∫ltiples condiciones.

```python
puntaje = 85

if puntaje >= 90:
    print("Excelente trabajo.")
elif puntaje >= 70:
    print("Buen trabajo.")  # Esta condici√≥n se cumple.
else:
    print("Necesitas mejorar.")
```

#### Operador ternario
Los operadores ternarios son m√°s conocidos en Python como expresiones condicionales. 

***Estructura:***
```python
condition_true if condition else condition_false
```
***Ejemplo:***
```python
# Operador ternario
edad = 18
mensaje = "Eres mayor de edad." if edad >= 18 else "Eres menor de edad."
print(mensaje)
```

### Bucles:
Los bucles te permiten repetir un bloque de c√≥digo varias veces.

#### Bucle `for`
El bucle `for` se utiliza para iterar sobre una secuencia (como una lista, tupla, diccionario o rango) y ejecutar un bloque de c√≥digo para cada elemento.

```python
frutas = ["manzana", "banana", "naranja"]
for fruta in frutas:
    print("Me gusta la " + fruta)
```

#### Bucle `while`
El bucle `while` se utiliza para ejecutar un bloque de c√≥digo mientras se cumpla una condici√≥n.

```python
contador = 0
while contador < 3:
    print(f"Contador: {contador}")
    contador += 1  # Incrementa el contador en 1.
```

### Control de bucles:
Estas palabras clave te permiten controlar el flujo de los bucles.

- `break`: Detiene el bucle por completo.
- `continue`: Salta a la siguiente iteraci√≥n del bucle.
- `pass`: Es una declaraci√≥n nula. No hace nada, pero se usa como marcador de posici√≥n.

#### Ejemplo de `break`:
```python
for i in range(10):
    if i == 5:
        break  # Detiene el bucle cuando i es 5.
    print(i)
```

#### Ejemplo de `continue`:
```python
for i in range(5):
    if i == 2:
        continue  # Salta la iteraci√≥n cuando i es 2.
    print(i)
```

#### Ejemplo de `pass`:
```python
for i in range(5):
    if i == 3:
        pass  # No hace nada cuando i es 3.
    print(i)
```
