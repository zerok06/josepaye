## ConcatenaciÃ³n de DataFrames

### Sentencia `concat()`
Combina DataFrames vertical u horizontalmente.

```python
import pandas as pd

df1 = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})
df2 = pd.DataFrame({'C': [7, 8, 9], 'D': [10, 11, 12]})

df = pd.concat([df1, df2], axis=1)

print(df)
```

### Sentencia `merge()`
Une DataFrames basados en una columna comÃºn.

```python
import pandas as pd

df1 = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

df2 = pd.DataFrame({'A': [1, 2, 3], 'C': [7, 8, 9]})


df = pd.merge(df1, df2, on='A', how='inner')

print(df)
```
> La propiedad how define el comportamiento de la operaciÃ³n de uniÃ³n. Puedes usar 'inner', 'left', 'right', o 'outer' para definir el comportamiento.
