## Reshaping de Arrays
El reshaping permite cambiar la forma de un array sin modificar sus datos. Esto es √∫til cuando necesitas reorganizar tus datos para an√°lisis o visualizaci√≥n.

```python
arr = np.array([1, 2, 3, 4, 5, 6])

# Cambiar su forma a una matriz 2x3
arr_reshaped = arr.reshape(2, 3)
print("Array redimensionado:\n", arr_reshaped)

# Cambiar su forma a una matriz 3x2
arr_reshaped = arr.reshape(3, 2)
print("Array redimensionado:\n", arr_reshaped)
```

## Aplanar Matrices
Aplanar una matriz significa convertirla en un array 1D. Esto es √∫til cuando necesitas simplificar la estructura de tus datos.

```python
matriz = np.array([[1, 2], [3, 4]])

matriz_aplanada = matriz.flatten()

print(matriz_aplanada) # Output: [1 2 3 4]
```
## Operaciones Diagonales
Las diagonales son elementos clave en muchas aplicaciones matem√°ticas. Con **NumPy**, puedes extraer la diagonal de una matriz o crear una matriz diagonal.

### Extraer la diagonal de una matriz
```python
matriz = np.array([[1, 2], [3, 4]])

diagonal = np.diag(matriz)

print(diagonal) # Output: [1 4]
```

### Calcular la traza (suma de la diagonal)
```python
matriz = np.array([[1, 2], [3, 4]])

traza = np.trace(matriz)

print(traza) # Output: 5
```
