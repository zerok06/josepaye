## Producto Punto

El producto punto es una operaciÃ³n matemÃ¡tica que se utiliza para calcular el producto de dos vectores. En **NumPy**, podemos realizar el producto punto utilizando la funciÃ³n `np.dot()`.

### Codigo de ejemplo en Python:

