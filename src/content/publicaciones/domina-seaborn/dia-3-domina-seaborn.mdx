## Bienvenida

¡Hola de nuevo! 🌟 Hoy aprenderemos a visualizar datos **categóricos**  usando Seaborn. Los datos categóricos son comunes en análisis de datos, y Seaborn ofrece herramientas especializadas para comparar grupos y categorías. ¡Comencemos!

## Gráficos de Barras Agrupadas (Count Plot)

Los gráficos de barras agrupadas son usados para comparar la distribución de variables categóricas.

```python
import seaborn as sns

# Datos
categorias = ['A', 'B', 'A', 'B', 'C', 'A', 'C']

# Crear el gráfico
sns.countplot(x=categorias)
```

## Gráficos de Caja por Categoría (Boxplot)

Los gráficos de caja por categoría son usados para comparar la distribución de variables numéricas en diferentes categorías.

```python
import seaborn as sns
import pandas as pd

# Datos
df = pd.DataFrame({
    'Categorías': ['A', 'A', 'B', 'B', 'C', 'C'],
    'Valores': [1, 2, 3, 4, 5, 6]
})

# Crear el gráfico
sns.boxplot(x='Categorías', y='Valores', data=df)
```

| Categorías | Valores |
| --- | --- |
| A | 1 |
| A | 2 |
| B | 3 |
| B | 4 |
| C | 5 |
| C | 6 |

Resultado de ejecución:
![grafico de caja por categoría](/assets/images/publicaciones/recursos/domina-seaborn/8.png)

## Gráficos de Barras Agrupadas (Bar Plot)

Los gráficos de barras agrupadas son usados para comparar la distribución de variables categóricas.

```python
import seaborn as sns
import pandas as pd

# Datos
df = pd.DataFrame({
    'Categorías': ['A', 'A', 'B', 'B', 'C', 'C'],
    'Valores': [1, 2, 3, 4, 5, 6]
})

# Crear el gráfico
sns.barplot(x='Categorías', y='Valores', data=df)
```
