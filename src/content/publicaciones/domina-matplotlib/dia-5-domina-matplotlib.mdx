## Grafico de caja (Boxplot)

Un boxplot muestra la distribuciÃ³n de una variable numÃ©rica. Puedes crear un boxplot utilizando la funciÃ³n `plt.boxplot()` en Matplotlib. Aqui tienes un ejemplo:


## Grafico de calor (Heatmap)

Un heatmap muestra la correlaciÃ³n entre dos variables numÃ©ricas.

```python
import matplotlib.pyplot as plt

# Datos
matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# Crear el heatmap
plt.imshow(matriz, cmap='hot', interpolation='nearest')

# PersonalizaciÃ³n

plt.xlabel('Variable 1')
plt.ylabel('Variable 2')

plt.show()
```

Resultado de ejecuciÃ³n:
![heatmap](/assets/images/publicaciones/recursos/domina-matplotlib/12.png)

> **Nota**: Podemos personalizar el heatmap con la funciÃ³n `plt.imshow()` para agregar una leyenda modificando el parÃ¡metro `cmap` y `interpolation`.
