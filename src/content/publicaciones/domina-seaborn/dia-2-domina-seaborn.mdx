## GrÃ¡ficos de DistribuciÃ³n (Distribution Plots)

### Histograma con KDE (Kernel Density Estimate)
El histograma muestra la frecuencia de los datos, mientras que el KDE estima la densidad de probabilidad.

```python
import seaborn as sns
# Datos
datos = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]

# Crear el grÃ¡fico
sns.histplot(datos, kde=True)
```

> EL KDE genera una curva que representa la densidad o suavisado de la distribuciÃ³n de los datos.

Puedes profundizar mas el tema de [KDE](https://cienciadedatos.net/documentos/pystats02-kernel-density-estimation-kde-python) (Kernel Density Estimate).

### Grafico de caja y violin (Boxplot and Violin Plot)

El boxplot muestra estadÃ­sticas clave (mediana, cuartiles, valores atÃ­picos), mientras que el violin plot combina un boxplot con una estimaciÃ³n de densidad.


```python

# Datos
datos = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4, 10]

# Boxplot

```
